# 阶段验收

## Phase 2

- 虚拟双摇杆可用
- 底盘模式只动底盘
- 云台模式只动云台
- 联动模式由右摇杆主控，左摇杆被忽略
- 松手、失焦、断连、心跳超时都归零
- 急停始终生效

## Phase 3

- YOLO 检测链路可单独启停
- 视频叠框正常
- 识别更新不阻塞视频预览

## Phase 4

- 只允许锁定 `person`
- 锁定后保存照片到 `runtime/media/locked_targets/`
- `index.json` 同步更新
- 媒体管理页可查看照片

## Phase 5

- 辅助跟随必须先锁定人物
- 云台优先跟踪
- 底盘只做低速辅助
- 丢失目标后进入搜索
- 丢失超过 10 秒后告警并停驶
- 急停可以打断跟随

## Phase 6

- 控制台保留 `场景操作` 入口
- `/scene` 页面可打开
- 页面明确提示“后续阶段开放”
- 不提供假任务或假执行结果
