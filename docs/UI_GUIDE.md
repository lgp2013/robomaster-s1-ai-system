> 文档版本：v9

## 1. 页面概述

RoboMaster S1 智能控制台是一个生产级遥控驾驶舱界面，采用深色工业控制台风格，视频画面是绝对主角。

### 视觉方向

- **风格**：深色工业控制台 + RoboMaster 机甲驾驶舱
- **主色调**：深色背景 + 高对比黄色安全色
- **字体**：Bahnschrift + Microsoft YaHei UI
- **布局**：视频优先 + 掌机式左右摇杆控制区

## 3. 各区域详细说明

### 3.1 顶部状态栏

| 元素 | 说明 |
|------|------|
| 品牌标题 | "RoboMaster S1 智能控制台" |
| 阶段说明 | 显示当前开发阶段，如"第二阶段 / 手动遥控与安全链路" |
| 重新扫描按钮 | 触发 ROS2 重新扫描 |
| 模拟视频按钮 | 切换到本地模拟视频源 |
| 媒体管理按钮 | 在新 Tab 打开媒体管理页面 |
| 全屏预览按钮 | 视频区域全屏显示 |

### 3.2 视频预览区域

- **位置**：页面左侧，占据主要空间
- **功能**：显示 MJPEG 视频流或模拟视频
- **叠加层**：YOLO 检测框（人物为黄色，障碍物为红色）
- **交互**：点击人物框可锁定目标（Phase 4）

### 3.3 系统状态面板

显示系统运行状态：

| 字段 | 说明 |
|------|------|
| ROS2 状态 | 已连接 / 未连接 |
| 视频状态 | 在线 / 离线 |
| FPS | 当前帧率 |
| 分辨率 | 视频分辨率 |
| 当前来源 | 视频源名称 |

### 3.4 机器人信息面板

显示机器人实时状态：

| 字段 | 说明 | 数据来源 |
|------|------|----------|
| 电池电量 | 电池百分比 | `/battery` |
| 底盘速度 | 线速度 / 角速度 | `/odom` |
| 云台偏航 | 当前偏航角度 | `/state` |
| 云台俯仰 | 当前俯仰角度 | `/state` |
| IMU 状态 | 惯性测量单元状态 | `/imu` |
| 连接质量 | 网络连接质量 | 系统检测 |
| 运行时长 | 累计运行时间 | 系统计算 |
| 错误码 | 当前错误码 | 暂不支持 |
| 固件版本 | 机器人固件版本 | 暂不支持 |
| 最后更新 | 数据刷新时间 | 系统生成 |

### 3.5 控制状态面板

显示控制链路状态：

| 字段 | 说明 |
|------|------|
| 控制链路 | WebSocket 连接状态 |
| 当前模式 | 底盘 / 云台 / 联动 |
| 底盘速度 | 当前底盘速度指令 |
| 云台速度 | 当前云台速度指令 |
| 急停状态 | 已触发 / 未触发 |

### 3.6 底部控制区

#### 模式切换

| 模式 | 说明 | 行为 |
|------|------|------|
| 底盘模式 | 只移动车辆，云台不动 | 左摇杆控制底盘，右摇杆无效 |
| 云台模式 | 只调整云台，车辆不移动 | 右摇杆控制云台，左摇杆无效 |
| 联动模式 | 车辆跟随云台方向 | 右摇杆主控，底盘跟随云台指向 |

#### 急停按钮

- **位置**：模式切换右侧，红色醒目按钮
- **行为**：点击后立即停止底盘和云台运动
- **解除**：点击"解除急停"按钮恢复控制

#### 虚拟摇杆

| 摇杆 | 位置 | 控制对象 |
|------|------|----------|
| 底盘摇杆 | 左侧 | 底盘移动（前后左右） |
| 云台摇杆 | 右侧 | 云台转动（偏航俯仰） |

## 4. 响应式布局

### 断点设计

| 分辨率 | 布局调整 |
|--------|----------|
| 1920×1080 | 双栏布局，视频区域最大化 |
| 1600×900 | 双栏布局，适当缩小间距 |
| 1366×768 | 双栏布局，侧边栏可折叠 |
| < 1200px | 单栏布局，侧边栏移到底部 |
| < 720px | 单列布局，摇杆缩小 |

### 字体自适应

使用 `clamp()` 函数实现字体大小自适应：

```css
h1 { font-size: clamp(1.1rem, 1.5vw, 1.6rem); }
```

## 5. 交互说明

### 5.1 视频源切换

1. 点击"重新扫描机器人"发现视频源
2. 系统自动连接默认视频源
3. 点击"使用本地模拟视频"切换到测试源

### 5.2 控制流程

1. 选择控制模式（底盘/云台/联动）
2. 拖动对应摇杆控制机器人
3. 松开摇杆自动归零
4. 点击急停立即停止

### 5.3 目标锁定（Phase 4）

1. 启用感知功能
2. 视频上显示检测框
3. 点击人物框锁定目标
4. 锁定后保存照片到媒体管理
